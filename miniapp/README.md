# 学吧 - 微信小程序

AI 驱动的智能学习助手小程序。

## 功能

- 📚 **项目管理**：创建学习项目，设定目标和截止日期
- 🌳 **知识图谱**：AI 自动生成知识点图谱，展示学习路径
- 📖 **教材生成**：为每个知识点生成详细教材内容
- 📝 **测验生成**：自动生成选择题和简答题，检验学习效果

## 页面结构

```
pages/
├── index/       # 首页 - 项目列表
├── project/     # 项目详情 - 知识图谱
└── study/       # 学习页 - 教材和测验
```

## 配置

修改 `app.js` 中的 `baseUrl` 指向你的后端地址：

```javascript
globalData: {
  baseUrl: 'http://localhost:8000',  // 改为你的后端地址
  token: ''
}
```

## 使用流程

1. 打开小程序，自动登录
2. 点击"+ 新建项目"创建学习项目
3. 进入项目，点击"生成知识图谱"
4. 选择知识点，生成教材和测验
5. 开始学习！

## 技术栈

- 微信小程序原生开发
- RESTful API 调用
- 支持 DashScope OpenAI 兼容模式
